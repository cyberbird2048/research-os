# AI Revolution Baseline

**This is an orientation skeleton, not a research report.** All content below is unverified framing carried over from the project handoff. It has not passed any review gate in `workflows/RESEARCH_WORKFLOW.md` and must not be cited as evidence. No investment conclusion may be drawn from this file.

## Purpose

Provide the shared reference frame that Phase 2 historical research will eventually map onto. This file does not assert that the AI revolution follows this pattern — it states the pattern to be tested, revolution by revolution.

## Unified Technological Revolution Lifecycle

```text
Technology breakthrough
  → Cost decline
  → Infrastructure buildout
  → Application explosion
  → Platform formation
  → Wealth migration
  → Bubble
  → Shakeout
  → New giants
```

Prior general-purpose technology revolutions (steam/rail, electricity/steel/oil/auto, semiconductors/PC/internet/mobile/cloud) are hypothesized to pass through this sequence, at different speeds, with different actors capturing value at each stage. This is a hypothesis to be tested per revolution via `templates/HISTORICAL_RESEARCH_TEMPLATE.md`, not an established law.

## Ten Standard Questions

Every historical revolution researched in this project must answer:

1. What is the technological breakthrough?
2. Why did it happen at this particular time?
3. Why did costs decline rapidly?
4. What is the biggest bottleneck?
5. Who sells the shovels (infrastructure)?
6. What were the first applications to scale?
7. How did wealth migrate across the value chain?
8. Who became the platform?
9. Where was the bubble?
10. Who survived after the bubble, and why?
