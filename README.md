# CyberBird Research OS

> A focused, versioned research operating system for discovering durable AI-era investment alpha through historical comparison, testable theses, and disciplined review.

## North Star

Improve one capability:

> Discover, explain, test, and track investment alpha created by the AI technological revolution.

## Repository Role

This repository is the **single source of truth** for durable research assets:

- frameworks
- patterns
- historical mappings
- investment theses
- predictions
- decisions
- reviews

It does **not** store raw news feeds, unprocessed PDFs, copied articles, or undigested notes.

## Current Scope

The current research scope is intentionally narrow:

1. Understand how previous technological revolutions created and redistributed wealth.
2. Map those recurring patterns to the AI revolution.
3. Convert the strongest mappings into testable investment theses.
4. Review and update those theses through evidence.
